# 🎮 RetroPie Gaming Device Project

A custom-built retro gaming console using **Raspberry Pi** and **RetroPie**, designed to emulate classic consoles like NES, SNES, Sega Genesis, PlayStation, and more. This project combines hardware setup, system configuration, and ROM management to deliver a plug-and-play retro gaming experience.

## 🛠️ Features

- Plug-and-play retro gaming console
- Pre-installed with RetroPie
- Supports multiple emulators (NES, SNES, Sega, PS1, etc.)
- Custom splash screen and UI themes
- USB game controller support
- Easy ROM addition via USB or network

## 💻 Tech Stack

- **Hardware**: Raspberry Pi 3/4, microSD card, HDMI cable, power adapter
- **Software**: RetroPie, Raspbian OS, EmulationStation
- **Tools**: Win32 Disk Imager / balenaEtcher, FileZilla, SSH

## 📦 Installation Steps

1. Flash RetroPie onto microSD using balenaEtcher or Win32 Disk Imager.
2. Insert SD card into Raspberry Pi and boot up.
3. Configure controllers and connect to Wi-Fi.
4. Transfer ROMs via USB drive or network.
5. Reboot and enjoy retro gaming!
